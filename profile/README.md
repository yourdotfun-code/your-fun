### What is your.fun?

your.fun is a platform where humans prove their humanity through behavioral biometrics and then learn with AI companions that adapt to their style. Everything is verified on Solana.

### Tech Stack

- **On-chain**: Solana (Rust / Anchor)
- **SDK**: TypeScript, Python
- **API**: Node.js / Express
- **Verification**: Behavioral fingerprinting + challenge-response

### Current Focus

Building out the Proof-of-Human protocol and AI companion learning system. Active on devnet.
